    Criar user numa so linha

    
sudo useradd -m -s /bin/bash nome_do_utilizador && sudo passwd nome_do_utilizador

    Apagar user

    
sudo userdel nome_do_utilizador

    Verificar tipo de arquivo

    
file ficheiro.txt

    Criar arquivo

    
touch novo_ficheiro.txt

    Cria um ficheiro com conteudo

    
echo "conteúdo do ficheiro" > ficheiro.txt


    Adicionar ao final do ficheiro:
    

echo "mais conteúdo" >> ficheiro.txt

    Move ou renomeia ficheiros

    
mv ficheiro.txt novo_nome.txt

    Ver tamanho de ficheiros

    
du -h ficheiro.txt

    Copiar um ficheiro para um novo local

    
cp ficheiro_origem.txt /caminho/para/destino/

    Copiar e renomear um ficheiro

    
cp ficheiro_origem.txt ficheiro_novo.txt

    Copiar um diretório inteiro

    
cp -r /caminho/do_diretorio_origem /caminho/para/destino/

    Processos em execução para o usuário atual

    
ps

    Listar todos os processos em execução no sistema

    
ps aux

    Terminar o processo com o ID de processo

    
kill 1234

    Encerrar processos com base em um nome

    
pkill firefox

    Executa um comando em segundo plano

    
comando &

    Retoma um processo suspenso em segundo plano

    
bg

    Exibe todas as interfaces de rede

    
ip a

    Mostra os endereços IP atribuídos à máquina

    
hostname -I

    Informacoes do sistema

    
uname -a

    Comprimir em TAR

    
tar -cvf nome_do_arquivo

    Comprimir em GZIP

    
gzip nome_do_arquivo

    Iniciar um servico de rede

    
sudo systemctl start nome_do_servico

exemplo: sudo systemctl start httpd

    Ver status do servico de rede

    
sudo systemctl status nome_do_servico
